What Is DOM And What Is The Purpose Of DOM?

The DOM, or Document Object Model, serves as a representation of a document's structure, allowing for manipulation of HTML tags and the provision of functionality to users.

How Will You Select HTML Elements Using DOM? Name The DOM Methods.

The DOM provides a mechanism for selecting HTML elements by their ID, class, or name.

  1. getElementsByClass(element)
  2. getElementById(element)
  3. getElementByTagName(element)
  4. querySelector(selector)
  5. querySelectorAll(selectors)

What Is Event Bubble?

Event Bubbling is a concept in the DOM (Document Object Model) where an element receives an event, and that event then bubbles up, or propagates, to its parent and ancestor elements in the DOM tree until it reaches the root element.

What Is A Callback Function And Why Will You Use It?

A callback function in JavaScript is a function passed as an argument to another function, to be executed later, often in response to an event or after completing an asynchronous operation. It's a fundamental concept used in event handling, asynchronous tasks, and functions like setTimeout() or setInterval().
